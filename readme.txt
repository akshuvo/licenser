== Description ==
Licenser is a WordPress plugin that allows you to manage licenses for your products. With Licenser, you can easily generate and validate license keys, track license activations, and provide updates to your customers.

== Installation ==
1. Upload the `licenser` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Licenser settings page and configure the plugin according to your needs.

== Frequently Asked Questions ==
= How can I generate license keys? =
To generate license keys, go to the Licenser settings page and click on the "Generate Keys" button. You can specify the number of keys to generate and customize the key format.

= How can I validate license keys? =
Licenser provides a simple API that you can use to validate license keys. You can find the API documentation on the Licenser website.
